Baseball statistic calculators

  • Win Expectancy Finder, which calculates the relative frequency of a team winning given their current situation (inning, outs, runners, score difference). You can also use the Win Expectancy Grapher to graph the win expectancy over an entire game, and visually see the key turning points of the game!
  • Expected Runs in an Inning, which calculates the expected number of runs scored in an inning given the current situation (outs, runners). It also shows the historical distribution of runs, which is helpful if you only need one run.
